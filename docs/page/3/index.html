Some of my Projects

All of the projects follow a Micro-Services Architecture.

  • Ceph-Chef is the Official Ceph Chef Cookbook used to install and manage Ceph. I created it at Bloomberg and open sourced it. I manage it at Ceph's Github Repo and we use it at Bloomberg to manage our large Ceph clusters.

    ceph-chef

    Ceph Chef Cookbook

  • AWS S3 is now considered the defacto Object Store standard. All Object Store products such as Ceph use the S3 SDK interface. Aws-sdk-rust is the S3 interface for Rust. It supports V2 and V3 Signatures so that all third party Object Stores can also be accessed.

    aws-sdk-rust

    AWS S3 SDK

  • S3LSIO is a powerful S3 utility that provides all of the S3 features of the AWS CLI and S3CMD plus benchmarking and other features you would find in part of JMeter but purpose built for you S3 environment. Also, there are no dependencies since it's built in Rust and it's very fast!

    s3lsio

    AWS S3 Utility

  • Klaus is a true Hybrid WebStack. It can be ran (with no dependencies) as a fast async I/O futures based Web Server, it can be ran as an API Server or as an Agent. It also has built-in support for AWS S3 and Redis.

    klaus

    Hybrid WebStack

  • Ceph-Rust is the Official Ceph Rust SDK for librados. It mirrors the `C` version of librados PLUS has Admin Socket features built-in. You can combine Ceph-Rust with Klaus to form a powerful and very fast backend doing Ceph maintenance. Core of Cepheus which is coming soon!

    ceph-rust

    Ceph Rust SDK

  • Rust Library for HTTP/2 which includes HPACK. It's the core of Klaus and has built-in logging.

    tokio-http2

    HTTP/2

  • Chef-BCS is the Official Bloomberg Cloud Storage (BCS) system built on Ceph-Chef. It will automatically build a Ceph Object Store cluster based on yaml data files. The default out-of-the box builds a 4 node Vagrant/VirtualBox Ceph Object Store on your laptop/desktop so that you can do development and tune before moving to your version of a staging environment.

    chef-bcs

    Chef-BCS

Ceph Librados for Rust

By Chris Jones on September 5, 2016

Offical Ceph Rust Librados Library The official Ceph librados Rust API has been released called ceph-rust. Ceph-rust can be found at crates.io and at https://github.com/ceph/ceph-rust. Ceph-rust is a very thin layer above the C librados library that drives Ceph. In addition, it has some higher level APIs that wrap the low-level C interface with Rust specific protection. Working with Ceph is now fast and safe.
